--- layout: default title: Simple Alpheios Enabled Site Template ---

This is a template that you can use to create a simple GitHub Pages website enhanced with embedded Alpheios Reading Tools.

Sections of Latin, Greek, Arabic or Persian text can be automatically enabled for double-click (Desktop)/long tap (Mobile) access from words to the Alpheios Reading Aids.

Double-click (if on a Desktop) or tap (if on a Mobile device) on the Latin and Greek words in the following sections to see how it works. Then view the Example HTML to see how to mark up the text. Follow the instructions at Getting Started to set up your own site.

Arma virumque canō, Trōiae quī prīmus ab ōrīs
Ītaliam, fātō profugus, Lāvīniaque vēnit
lītora, multum ille et terrīs iactātus et altō
vī superum saevae memorem Iūnōnis ob īram;
